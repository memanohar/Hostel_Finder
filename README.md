# Hostel_Finder
Hostel Finder is a web application that helps students quickly discover nearby hostels based on location, budget, and basic amenities. It aims to remove offline broker dependency and make the search process faster and more transparent.​

#Features
Search hostels by location and price range for easier decision making.​
View hostel details like photos, facilities, and contact information in one place.​
Simple interfaces for students and hostel owners to register and manage listings.​

#Tech Stack
Frontend: (e.g., React / HTML, CSS, JavaScript)
Backend: Node.js, Express, MongoDB
